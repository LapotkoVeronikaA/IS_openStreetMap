О проекте

Данный веб-ресурс представляет собой централизованный реестр организаций и их структурных подразделений, разработанный на примере ЧОУ ВО «Московский университет им. С.Ю. Витте».

Основные возможности:

  • Централизованный реестр: Ведение единой базы данных по всем учебным корпусам, филиалам и другим структурным единицам.
  • Интерактивная карта: Визуализация расположения всех объектов на карте для удобной навигации и анализа.
  • Детальная информация: Хранение подробных данных по каждой организации, включая адреса, контакты, структуру и фотоматериалы.
  • Управление доступом: Гибкая система ролей и прав для разграничения доступа к информации и функциям управления.
  • API для интеграции: Возможность интеграции с другими информационными системами для обмена данными.

Проект разработан в рамках преддипломной практики с использованием современных веб-технологий, включая Python, Flask, SQLAlchemy и Bootstrap 5.
